# delete\_all\_cookies

## Syntax

```python
py.delete_all_cookies()
```

## Usage

{% code title="correct usage" %}
```python
py.delete_all_cookies()
```
{% endcode %}

{% code title="incorrect usage" %}
```python
# Errors, 'delete_all_cookies' yields None
py.delete_all_cookies().get()
```
{% endcode %}

## Arguments

* None

## Yields

* None
