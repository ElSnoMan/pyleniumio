# is\_enabled

## Syntax

```python
Element.is_enabled() -> bool
```

## Usage

{% code title="correct usage" %}
```python
py.get("#button").is_enabled()
```
{% endcode %}

## Arguments

* None

## Yields

* <mark style="color:orange;">**bool**</mark> - True if the element is enabled, else False
