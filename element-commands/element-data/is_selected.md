# is\_selected

## Syntax

```python
Element.is_selected() -> bool
```

## Usage

{% code title="correct usage" %}
```python
py.get(".option").is_selected()
```
{% endcode %}

## Arguments

* None

## Yields

* <mark style="color:orange;">**bool**</mark> - True if the element is selected, else False
