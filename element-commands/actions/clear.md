# clear

## Syntax

```python
Element.clear()
```

## Usage

{% code title="correct usage" %}
```python
py.get("input").clear()
```
{% endcode %}

## Arguments

* None

## Yields

* <mark style="color:orange;">**Element**</mark> - The current Element that was cleared so you can chain commands.
