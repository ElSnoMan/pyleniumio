# right\_click

## Syntax

```text
Element.right_click()
```

## Usage

{% code title="correct usage" %}
```python
py.get('#context-menu').right_click()
```
{% endcode %}

{% code title="incorrect usage" %}
```python
# 'py' does not have this command
py.right_click()
```
{% endcode %}

## Arguments

* None

## Yields

* **\(Pylenium\)** so you can chain another command
