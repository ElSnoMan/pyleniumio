# window\_size

## Syntax

```python
py.window_size
```

## Usage

{% code title="correct usage" %}
```python
size = py.window_size

# print the width
print(size['width'])

# print the height
print(size['height']
```
{% endcode %}

## Arguments

* None

## Yields

* **Dict\[str, int\]** The current window's size as a dictionary
