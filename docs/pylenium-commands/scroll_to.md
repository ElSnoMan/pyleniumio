# scroll\_to

## Syntax

```text
py.scroll_to(x, y)
```

## Usage

{% code title="correct usage" %}
```python
# scroll down 500px
py.scroll_to(0, 500)
```
{% endcode %}

## Arguments

* **`x (int)`**: The number of pixels to scroll horizontally
* **`y (int)`**: The number of pixels to scroll vertically

## Yields

* **\(Pylenium\)** so you can chain another command
